# React Three Fiber Intro – Day 3 🚀

This project is part of my **React Learning Series**, where we explore different concepts of React step-by-step.

In **Day 3**, we move beyond traditional UI and explore how **React can also be used to create 3D experiences** using **React Three Fiber**.

Instead of building only buttons, forms, and layouts, we create a **simple and elegant 3D scene inside a React application**.

---

## 🎥 About the Video

In this video we learn how to integrate **Three.js with React** using **React Three Fiber**, and create our **first animated 3D object**.

The project demonstrates a **clean hero section with a rotating 3D cube**, styled with **Tailwind CSS**.

This helps beginners understand that **React is not limited to 2D interfaces — it can also power immersive 3D web experiences.**

---

## 📚 Topics Covered

This project explains the following core concepts:

- Introduction to **Three.js**
- What is **React Three Fiber**
- Setting up **React Three Fiber in a React project**
- Using the **Canvas** component
- Creating a **3D object with Mesh**
- Understanding **Geometry and Material**
- Adding **Lights in a 3D scene**
- Animating objects using **useFrame**
- Adding interaction using **OrbitControls**
- Styling the layout using **Tailwind CSS**
