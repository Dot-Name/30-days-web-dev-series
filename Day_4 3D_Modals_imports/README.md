# Day 4 – Adding 3D Models in React Three Fiber

Welcome to **Day 4** of the **30 Days Web Development Series** 🚀

In this project, we learn how to **import and display 3D models inside React using React Three Fiber**.

Until now we created simple 3D shapes like boxes and spheres.  
But in real applications we usually use **ready-made 3D models created in tools like Blender**.

In this tutorial we will load a **GLB model** and render it inside a React application.

---

# What You Will Learn

• What 3D models are  
• Common 3D model formats (GLB / GLTF)  
• How to download free 3D models  
• How to load models using `useGLTF()`  
• How to render models using `<primitive />`  
• How to add lights in a 3D scene  
• How to control the camera using `OrbitControls`
